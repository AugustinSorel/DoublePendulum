# Simple Double Pendulum Simulation

## Default Value

* θ1 = π / 2 
* θ2 = π
* circle1 mass = 10
* circle2 mass = 10
* gravity = 1
* friction = 0

## Features

* Change the weight of the circles
* Change the length of the arms
* Trace the old positions
* Reset the simulation
* Pause/Resum
* Full Screen Mode
* Change the frictopn
* Change the gravity
